### Tech Stack
- HTML5 (Semantic Structure)
- CSS3 (Glassmorphism & Advanced Animations)
- Vanilla JavaScript (State-driven UI Logic)

### Key Features
- **Three Game Modes:** Normal, Speed, and Survival.
- **Intelligent Scoring:** Rewards speed and consecutive streaks.
- **Dynamic Quiz Engine:** Randomizes a 100-question bank into 15-question sessions.
- **Performance Analytics:** Accuracy percentage and peak streak tracking.
